Editor di testo

  • cp r0 r1: mette v1 (valore o registro) in r0
  • add r0 v1: r0 + v1 (valore o registro)
  • sub r0 v1: r0 - v1 (valore o registro)
  • mul r0 v1: r0 * v1 (valore o registro)
  • div r0 v1: r0 / v1 (valore o registro)
  • load r0 m0: carica il valore della memoria m0 nel registro r0
  • store m0 r0: salva il valore del registro r0 nella memoria m0
